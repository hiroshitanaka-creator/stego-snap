• AI vision can see the grid size, but still struggles to decode accurately
• This proves AI hallucinates based on context rather than executing the logic

💡 How it works:
1. Text → Binary (8 bits per ASCII character)
2. Binary → Grid of black (1) and white (0) pixels
3. Grid size is explicitly written in the header
4. Decoding requires systematic pixel reading, not just "looking"
